함수 표현식

  • 익명 함수:이름 없이 선언한 함수

    function(매개변수) { 명령 }

  • 즉시 실행 함수:함수를 정의하는 것과 동시에 실행

    (function() { 명령 }());

    (function(매개변수) { 명령 } (인수));

  • 화살표 함수:화살표 기호(=>) 왼쪽에는 매개변수를, 오른쪽에는 함수를 작성함. 익명 함수일 경우에만 사용할 수 있음.

    (매개변수) => { 함수 내용 }
