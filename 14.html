변수

변수란? 프로그램을 실행하는 동안 값이 여러 번 달라질 수 있는 데이터를 말합니다.

변수 작성 규칙
1.변수의 이름은 대소 문자를 구별하여 작성합니다.
2. 영문자와 숫자, 언더스코어(_)만 사용하며 숫자로 시작할 수 없습니다.

변수 선언 방법: var [변수명]

자료형

종류 설명
기본유형 숫자형 따옴표 없이 숫자로만 표시합니다.
문자열 작은따옴표('')나 큰따옴표("")로 묶어서 나타냅니다.
논리형 참(true)과 거짓(false)이란 2가지 값만 가집니다.
복합유형 배열 하나의 변수에 여러 개의 값을 저장합니다.
객체 함수와 속성들이 함께 포함된 자료형입니다.
특수 유형 undefined 데이터 유형이 지정되지 않았을 때의 상태를 나타냅니다.
null 데이터 값이 유효하지 않은 상태를 나타냅니다.

연산자

종류 설명
산술 연산자 +, -, *, /, %, ++, --
할당 연산자 =, +=, -=, *=, /=, %=
비교 연산자 ==, !=, ===, !==, <, <=, >, >=
논리 연산자 !, &&, ||
조건 연산자 (조건) ? true일때 실행할 명령 : flase일 때 실행할 명령
